# PassGuard

PassGuard is a password manager application designed to securely store and manage user passwords. The PassGuard API allows developers to interact with the PassGuard application, managing user accounts and securely storing and retrieving password entries. This API is built to ensure secure and efficient operations while providing a straightforward interface for clients.
 Project was done by <a href="https://https://github.com/Techdockttor/" target="_blank">Diogu NkeJesus</a>, <a href="https://https://github.com/Melusi2022/" target="_blank"> Lindani Melusi Khumalo.</a>

## Technologies Used

- **Backend**: Node.js, Express
- **Database**:  MongoDB
- **Encryption**:  bcrypt, AES (Advanced Encryption Standard)
- **Version Control**:  Git and GitHub

### ðŸŒŸ Overview of the Backend Architecture of PassGuard
The PassGuard backend is built using Node.js and Express, with MongoDB as the database. It handles password management and user authentication, ensuring security and scalability.

- **Node.js and Express**: manage HTTP requests, routes, and middleware.
- **MongoDB**: stores user data and passwords in a flexible, schema-less format, with Mongoose used for data modeling.
- **JWT**: secures API endpoints, while bcrypt hashes passwords before storage.
- **API Endpoints**: provide RESTful services for operations like user login and password management.
- **Logging tracks key activities, and centralized error handling**: ensures robust security and performance.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Techdockttor/PassGuard1.git
   ```
2. Navigate to the project directory:
   ```bash
   cd PassGuard1
   ```
3. Navigate to the project directory:
   ```bash
   cd backend
   ```
4. Install the required dependencies:
   ```bash
   npm install
   ```
   
## Development

5. Run the application:
   ```bash
	npm start
   ```
6. Open your web browser and visit `http://localhost:4000`

## Production

7. shut the application    
	```bash
	Ctrl C
   ```
